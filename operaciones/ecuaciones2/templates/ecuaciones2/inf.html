Ecuaciones de Segundo Grado

Una ecuación de segundo grado (o ecuación cuadrática) es una expresión matemática donde la variable aparece elevada al cuadrado. Su forma general es:

ax² + bx + c = 0

Donde:
• a, b y c son números reales.
• a ≠ 0

Fórmula general

x = (-b ± √(b² − 4ac)) / 2a

Pasos para resolver

  1. Identificar a, b y c.
  2. Calcular el discriminante:
    • Δ > 0 → Dos soluciones reales
    • Δ = 0 → Una solución real
    • Δ < 0 → Soluciones complejas
  3. Aplicar la fórmula general.
  4. Obtener x₁ y x₂.

Ejemplo

2x² + 3x - 5 = 0

Discriminante:

Δ = 49

Soluciones:

x₁ = 1
x₂ = -2.5
