INFORMACIÓN SOBRE LOS PERÍMETROS

¿Qué es el perímetro?

El perímetro es la suma de las longitudes de todos los lados de una figura geométrica. Sirve para conocer la medida del borde o contorno de una forma.

¿Para qué sirve el perímetro?

  • Medir bordes de terrenos, figuras o superficies.
  • Calcular cercas o delimitaciones.
  • Resolver problemas de geometría básica.

Pasos generales para calcular un perímetro

  1. Identificar la figura geométrica.
  2. Determinar las medidas de sus lados.
  3. Aplicar la fórmula correspondiente.
  4. Sumar todos los lados.

Perímetro de un Cuadrado

Fórmula: P = 4 × lado

  1. Mide un lado del cuadrado.
  2. Multiplica esa medida por 4.
  3. El resultado es el perímetro.

Perímetro de un Rectángulo

Fórmula: P = 2 × (base + altura)

  1. Mide la base del rectángulo.
  2. Mide la altura.
  3. Suma base + altura.
  4. Multiplica el resultado por 2.

Perímetro de un Triángulo

Fórmula: P = lado1 + lado2 + lado3

  1. Mide los tres lados del triángulo.
  2. Suma las tres longitudes.
  3. El total es el perímetro.

Ejemplo rápido

Un rectángulo con base 5 cm y altura 3 cm:
P = 2 × (5 + 3) = 2 × 8 = 16 cm

Perímetro de un Círculo

El perímetro de un círculo también se conoce como circunferencia. Se calcula usando el número π (pi), que aproximadamente vale 3.1416.

Fórmula:

P = 2 × π × r
donde r es el radio del círculo.

  1. Mide el radio del círculo.
  2. Multiplica el radio por 2.
  3. Multiplica el resultado por π.
  4. El resultado es el perímetro del círculo.

Ejemplo:
Si el radio es 4 cm:
P = 2 × 3.1416 × 4 = 25.13 cm
